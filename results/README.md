# Results Summary

This folder contains visual summaries of the experimental and production results.

All plots are generated using:

`generate_results_plots.py`

## To regenerate results:

```bash
cd results
python generate_results_plots.py
```

---

## Included Visualizations

1. UCI HAR split comparison (80:20, 70:30, 60:40)
2. UCI vs RealWorld (8 Users) comparison
3. Window-level accuracy comparison (8 vs 5 users)
4. Window-level vs file-level accuracy
5. Window size impact analysis

These plots summarize the core findings of the research and deployment phases.
